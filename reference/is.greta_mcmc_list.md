# Is object a `greta_mcmc_list`?

Is object a `greta_mcmc_list`?

## Usage

``` r
is.greta_mcmc_list(x, ...)
```

## Arguments

- x:

  An object that may be a greta_mcmc_list

- ...:

  extra args (not currently used)

## Value

logical TRUE/FALSE

## Examples

``` r
is.greta_mcmc_list(1)
#> [1] FALSE
```
