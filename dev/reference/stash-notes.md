# Retrieve python messages.

These functions retrieve specific python error messages that might come
up during greta use.

## Usage

``` r
greta_notes_tf_num_error()
```

## Value

Invisibly returns `NULL`; called for its side effect of printing the
stored message.

## Examples

``` r
if (FALSE) { # \dontrun{
greta_notes_tf_num_error()
} # }
```
